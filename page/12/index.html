imports

public interface UserConvert {

mysql事务、锁、隔离级别

mysql事务

事务ACID原则

原子性(Atomicity)

事务是数据库的逻辑工作单位，它对数据库的修改要么全部执行，要么全部不执行。

一致性(Consistemcy)

事务前后，数据库的状态都满足所有的完整性约束。

隔离性(Isolation)

并发执行的事务是隔离的，一个不影响另一个。
如果有两个事务，运行在相同的时间内，执行相同的功能，事务的隔离性将确保每一事务在系统中认为只有该事务在使用系统。
通过设置数据库的隔离级别，可以达到不同的隔离效果。

持久性(Durability)

在事务完成以后，该事务所对数据库所作的更改便持久的保存在数据库库中。

创建主题，同一个主题创建一次即可。

.\bin\windows\kafka-topics.bat –create –zookeeper localhost:2181 –replication-factor 1 –partitions 1 –topic test1
或
.\bin\windows\kafka-topics.bat –create –bootstrap-server localhost:9092 –replication-factor 1 –partitions 1 –topic test1

启动生产者和消费者

.\bin\windows\kafka-console-producer.bat –broker-list localhost:9092 –topic test1

.\bin\windows\kafka-console-consumer.bat –bootstrap-server localhost:9092 –topic test1 –from-beginning

列出所有topic

.\bin\windows\kafka-topics.bat –list –bootstrap-server localhost:9092

查看指定topic

.\bin\windows\kafka-topics.bat –describe –bootstrap-server localhost:9092 –topic test1

增加topic的partition

.\bin\windows\kafka-topics.bat –bootstrap-server localhost:9092 –alter –topic test1 –partitions 5

.查看 topic 指定分区 offset 的最大值或最小值

.\bin\windows\kafka-run-class.bat kafka.tools.GetOffsetShell –topic test1 –time -1 –broker-list 127.0.0.1:9092 –partitions 0

删除topic

.\bin\windows\kafka-topics.bat –bootstrap-server localhost:9092 –topic test2 –delete

windows环境下删除topic需要关注brokder情况

消费消息

1.从头开始
.\bin\windows\kafka-console-consumer.bat –bootstrap-server localhost:9092 –topic test1 –from-beginning
2.从尾部开始
从尾部开始取数据，必需要指定分区：
.\bin\windows\kafka-console-consumer.bat –bootstrap-server localhost:9092 –topic test1 –offset latest –partition 0

  1. 指定分区
    .\bin\windows\kafka-console-consumer.bat –bootstrap-server localhost:9092 –topic test1 –offset latest –partition 0

消费者Group

  1. 指定Group
    .\bin\windows\kafka-consumer-groups.bat –bootstrap-server localhost:9092 –topic test1 -group test_group –from-beginning
  2. 消费者Group列表
    .\bin\windows\kafka-consumer-groups.bat –bootstrap-server localhost:9092 –list
  3. 查看Group详情[用来判断是否有延迟数据]
    .\bin\windows\kafka-consumer-groups.bat –bootstrap-server localhost:9092 –group test_group –describe
  4. 删除group
    .\bin\windows\kafka-consumer-groups.bat –bootstrap-server localhost:9092 –group test_group –delete
