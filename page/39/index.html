git克隆、更新、提交

发表于 2017-07-02 | 分类于 git ， 笔记
字数统计: 1.4k | 阅读时长 ≈ 1 mins.

克隆

git clone

可以直接克隆分支

  1. git clone git地址 分支号
    如：git clone git@git.ch-ins.com.cn:xx/xx.git branchName
  2. git clone -b 分支号 git地址
    如：git clone -b branchName git@git.ch-ins.com.cn:xx/xx.git

    先克隆主干，再切到分支

  3. git clone 主干地址
  4. 查看主干与分支名，git分支跟踪关系
    git branch -a
  5. 切换到分支，一般切换过去自动更新代码，如果不放心，可以再pull
    git checkout branchName

更新

git pull、git fetch

  1. git pull ，默认是 git pull -merge 更新后自动合并，且合并后会产生额外的commit
  2. git pull –rebase是以rebase的形式获取合并分支，不会产生额外的commit
  3. git fetch ， 只获取，需手动合并

提交

git add 、 git commit、git push

  1. 通常情况下先使用git status查看未提交代码
    可以使用git add . 追加所有未提交文件，或者使用git add 文件 追加指定文件
  2. 提交追加文件到本地仓库
    git commit -m “注释”
  3. git push提交远程仓库
  4. git commit –amend 撤销
