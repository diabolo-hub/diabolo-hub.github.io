发表于 2022-03-09 | 分类于 hexo ， 笔记
字数统计: 1.6k | 阅读时长 ≈ 2 mins.

https://hexo.io/zh-cn/docs/commands.html

  1. hexo n(new) fileName # 新建一篇文章。如果没有设置 layout 的话，默认使用 _config.yml 中的 default_layout 参数代替。如果标题包含空格的话，请使用引号括起来。 hexo g(generate) # 生成网页
    hexo s(server) # 启动服务器
    hexo d(deploy) # 部署到线上
    hexo p(publish) # 发表草稿。
    hexo version # 显示hexo版本
    hexo –debug # 调试模式，在终端中显示调试信息并记录到 debug.log
    hexo server –config custom.yml # 使用 custom.yml 代替默认的 _config.yml
    hexo clean # 清除缓存文件 (db.json) 和已生成的静态文件 (public)。 在某些情况（尤其是更换主题后），如果发现您对站点的更改无论如何也不生效，您可能需要运行该命令。
    hexo –draft # 显示 source/_drafts 文件夹中的草稿文章。
    hexo generate –config custom.yml,custom2.json,custom3.yml # 使用 custom.yml, custom2.json 和 custom3.yml，其中 custom3.yml 优先级最高，其次是 custom2.json
