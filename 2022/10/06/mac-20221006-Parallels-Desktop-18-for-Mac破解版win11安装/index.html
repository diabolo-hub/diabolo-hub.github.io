每次需要以此种命令运行parallels desktop: sudo -b /Applications/Parallels\ Desktop.app/Contents/MacOS/prl_client_app

回车后会提示输入密码，输入Mac系统密码然后按回车，这里输入密码不会显示，输完直接按回车即可。然后在弹出窗口按提示打开所有权限，如果出现安装Windows的界面，点击左下角的“打开”，在“ 用户名 - 文稿 - parallels ”目录下面找到你原来的Windows系统打开，就可以解决Parallels Desktop 16不能联网的问题了

pd虚拟机不能联网怎么办？如何解决Parallels Desktop虚拟机不能联网的问题

  1. 将windows关机
  2. 通过sudo -b 启动
  3. 重新打开windows，通过左下角打开按钮来打开新的win窗口
  4. 给windows当前版本添加快照，右键windows -> 操作 -> 拍快照
  5. 重新安装 Parallels Tools，右键windows -> 操作 -> 重新安装 tools
  6. 共享网络（建议），可以ping通，但是浏览器无法访问，打开shadowly可以访问（公司网络ip地址限制）
  7. 关掉shadowly之后，浏览器也可以访问网络（暂不明白什么情况）
